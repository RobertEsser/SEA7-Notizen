Notizen
Git-Kommandos:
init →GIT – Initalisierung des jeweiligen Verzeichnisses
 add → unter Angabe des Dateinamens zur Aufnahme im GIT
commit -m „Kommentar“ , zur Bestätigung

weitere Kommandos:
status, log,  checkout, branch
checkout -b „Name Branch“ zum Branchwechsel
merge - zusammenführen

Einmalige GITHUB Einrichtung
Kommandos im Terminal::
eval $(ssh-agent -s)
ssh -add
Verzeichnis für Synchronisation anlegen
git clone "ssh-Key aus Github" (liegt im Github -Seite Repository unter Code)

git fetch - zwecks sync
git branch -a - zeigt alle Branches
git checkout --track "Name Branch mit Remotepfad"
